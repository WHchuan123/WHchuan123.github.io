题意

在一个nnn维有限超立方体((1,p1),(1,p2),...,(1,pn))​((1, p_1), (1, p_2), ... , (1, p_n))​((1,p1​),(1,p2​),...,(1,pn​))​中，你可以在若干个位置放一只萤火虫。

萤火虫可以行动若干次，每一次从(x1,x2,x3,...,xn)(x_1, x_2, x_3,...,x_n)(x1​,x2​,x3​,...,xn​)走向(y1,y2,y3,...,yn)(y_1, y_2, y_3,...,y_n)(y1​,y2​,y3​,...,yn​)，都要满足：

1.起点和终点要在超立方体内；

2.∀ixi≤yi\forall_i x_i \le y_i∀i​xi​≤yi​；

3.(∑yi−xi)=1(\sum y_i - x_i) = 1(∑yi​−xi​)=1。

设最开始最少放kkk个，才能确保存在一种方案，对于每一个单位空间，至少用一只萤火虫遍历它，求放的数量最少的方案数。

题解

最优解的结构可以用Dilworth定理来分析。

Dilworth定理表明，最小链覆盖数等于最大反链数，也即选出一个集合，使得集合中的位置两两不可达，要最大化这个集合的大小。设所有极大集合中大小最大的为sss，那么最优方案一定是放了sss个萤火虫的。

如果我们把所有解和选出反链集合作一个一一映射，最优解的反链集合一定是极大的。

而Sperner定理表明，若要选择一个集合SSS的幂集的一个子集使得其中没有一个集合包含在另一个集合中，则这种子集的大小最大为(∣S∣⌊∣S∣/2⌋)\binom {|S|}{\lfloor |S| / 2 \rfloor}(⌊∣S∣/2⌋∣S∣​)。

则Sperner定理的一个推广恰好能给出本问题的解：

选择nnn维坐标之和为M=⌊12∑i=1n(pi+1)⌋M = \lfloor \frac {1}{2} \sum_{i = 1} ^ n (p_i + 1) \rfloorM=⌊21​∑i=1n​(pi​+1)⌋的位置即可达到最大的数量。

则现在问题转化为求∑i=1nxi=M\sum_{i = 1} ^ n x_i = M∑i=1n​xi​=M且1≤xi≤pi1 \le x_i \le p_i1≤xi​≤pi​的解的数量。

这是个经典问题，可以用容斥来做。

但考虑到n≤32n \le 32n≤32，不能直接容斥。
